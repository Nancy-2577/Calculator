# Calculator
Basic calculator program in C

## Description
This is a basic calculator program written in C language. 
It performs simple arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- modulus

## How to Run
1. Open the program in any C compiler (like Turbo C, CodeBlocks, or VS Code)
2. Compile the program
3. Run the executable file
